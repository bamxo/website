Wildin Foos

A Unity Project

Wildin Foos is the video game is one I'm currently working on and is my first game with a large scope that involves many components.

The basic concept of the game is a co-op survival/roguelike styled game where players are placed on an island and the goal is to defeat 3 unique bosses and collect their gemstones so they can be placed on an escape boat where they defeat 1 final grand boss to be set free.

In order to achieve these concepts, the game engine will utilize procedural generation to randomly create a terrain. The game would also incorporate many 3D Assets to help make it visually stunning. Wildin Foos would also have an enticing story to go along with the gameplay.

Mediverse

A Web Application

Mediverse is an early-stage web application designed to streamline the management of Electronic Health Records (EHR). As the project lead, I oversaw the entire development process, working closely with a diverse team that includes another developer, a graphic designer, and a healthcare consultant.

Key features of Mediverse include secure user authentication and real-time data management, powered by Firebase. The platform currently supports a user-friendly homepage, along with fully functional signup and login systems. We've also developed a mockup of the admin dashboard, setting the foundation for a comprehensive EHR solution that will enhance efficiency and accuracy in healthcare settings.

This project is ongoing, with plans to expand functionality and further refine the user experience to meet the rigorous demands of the healthcare industry.
